# **App Name**: FormaAfrique

## Core Features:

- Course Catalog: Browse a list of available courses with descriptions.
- Course Details: View detailed information about a specific course.
- Video Module Player: Stream video lessons as part of a course module.
- User Authentication: Secure user login and registration using Firebase Authentication.
- User Dashboard: Allow users to manage profile information, track learning progress.
- AI Tutor Chatbot: A virtual AI tool to assist learners.
- Database Integration: Utilize Firestore to store all of the courses.

## Style Guidelines:

- Primary color: #1E3A8A (a saturated blue) for trust, education.
- Background color: #E0E7FF (very light desaturated blue).
- Accent color: #A3E635 (a contrasting lime green).
- Body and headline font: 'PT Sans', a modern and friendly sans-serif, suitable for both headlines and body text.
- Use modern and clear icons.
- Clean, modern, and responsive layout.
- Subtle animations and transitions for enhanced user experience.
